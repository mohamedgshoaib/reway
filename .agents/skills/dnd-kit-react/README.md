# dnd-kit/react Skill Package

This package is designed for LLM agents implementing modern `@dnd-kit/react` drag-and-drop interfaces.

## Contents
- `SKILL.md`: compact operational guide under the 200-line rule.
- `references/01-core-api.md`: provider, hooks, overlay, utilities.
- `references/02-sortable-patterns.md`: single-list, kanban, external state.
- `references/03-sensors-collisions-modifiers-feedback.md`: advanced tuning.
- `references/04-agent-checklists.md`: production and debugging checklists.
- `examples/SortableList.tsx`: one-list sortable implementation.
- `examples/KanbanBoard.tsx`: multi-list kanban implementation.

## Source basis
Built from the official dnd kit React documentation pages supplied by the user.
