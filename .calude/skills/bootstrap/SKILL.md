# Bootstrap

Run this once at the start of every fresh repository.

1. Read spec/index.md and spec/skills.md. Note that both are empty templates.
2. Invoke grill-me. Interview the user one question at a time until you have enough truth to fill both files completely. Do not rush. Do not assume.
3. Draft spec/index.md. Show it to the user. Wait for explicit approval before saving.
4. Draft spec/skills.md based on the installed skills in .agents/skills/ and what was discussed. Show it. Wait for approval before saving.
5. Confirm: "Bootstrap complete. Both spec files are approved and saved. Ready to build."

Do not write any code during this command. Do not make implementation decisions.
